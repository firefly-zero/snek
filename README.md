# Snek

Snake-inspired game with 360Â° movement. Written in Go for [Firefly Zero](https://fireflyzero.com/) using [the official SDK](https://github.com/firefly-zero/firefly-go).

## Installation

Install from the [catalog](https://catalog.fireflyzero.com/) using [firefly-cli](https://github.com/firefly-zero/firefly-cli):

```bash
firefly_cli import lux.snek
```

Build and install from source:

```bash
git clone https://github.com/firefly-zero/snek.git
cd snek
firefly_cli build
```
